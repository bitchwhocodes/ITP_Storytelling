Welcome to the faceshift Unity plugin!
======================================

An example scene can be found under "Animation/Examples/Scenes".
It contains a model prefab ready to be animated with the faceshift plugin.

For a complete tutorial on how to use the faceshift plugin, please visit http://doc.faceshift.com/plugins/unity.html.

For more help concerning faceshift Studio please visit http://doc.faceshift.com/studio.

version 1.0
===========

This plugin allows you to import clips from faceshift studio and animate models in Unity.

It will allow you to:
- Load clips from faceshift studio
- Perform retargeting on Unity models
- Create animations
